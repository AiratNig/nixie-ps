# Nixie power supply

High voltage flyback boost converter for nixie tubes.

* **V_in** = 5V - 12V
* **V_out** = 150V - 200V
* **P_out** = 4W @ 180V from 5V, 8W @ 180V from 12V

** BOARD IS NOT TESTED YET **

This specifications are goals. Table will be updated after board test.
